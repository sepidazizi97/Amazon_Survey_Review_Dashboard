# Amazon Transportation Survey Dashboard

This project analyzes transportation survey results from employees at the Amazon facility. The dashboard was developed to support Ben Franklin Transit in understanding commute patterns, transportation barriers, transit interest, and potential service or employer-based mobility improvements.

## Project Goals

- Summarize each survey question separately
- Visualize commute behavior and transportation needs
- Identify themes in open-ended survey responses
- Explore relationships between survey questions
- Support transit planning and employer transportation strategies

## Tools Used

- Python
- Google Colab
- Pandas
- Streamlit
- Plotly
- Excel
